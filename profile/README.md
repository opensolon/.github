## 特性:


| 特性      |  描述 | 
|---------| -------------------- | 
| 更高的计算性价比 | 并发高 2～ 3 倍；内存省 50%     |
| 更快的开发效率 | 内核小，入门快；调试重启可快至 10 倍     |
| 更好的生产与部署体验 | 打包最多缩小 90%     |
| 更大的兼容范围 | 非 java-ee 架构；同时支持 java8 ～ java22，graalvm native image      |
